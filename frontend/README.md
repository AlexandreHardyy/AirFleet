## How to run

- Install dependencies
```
flutter pub get
```

- Run the app
```
flutter run
```

- Run the web app on Chrome
```
flutter run -d chrome
```

- Run the web app on Arc
```
flutter run -d arc
```

- Run tests
```
flutter test
```

- Update swagger here: http://localhost:3001/swagger/index.html
